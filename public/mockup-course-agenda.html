NOUNIT NAMEOBJECTIVETYPE
Unit 1Understand the Fundamentals of Genero MobileBy the end of this unit, learners will be able to explain what Genero Mobile applications are, describe how they fit within the Four Js development ecosystem, and summarize the purpose of using a single code base for multi-platform mobile development.Cert-Critical
Unit 2Responsive LayoutIdentify the three layout containers — VBox, HBox, and Grid. Apply the STRETCH=X property. Recognise how Genero classifies screens into Small, Medium, and Large. Use HIDDEN@screen-size and ORIENTATION@screen-size.Supporting
Unit 3Recognise Key Mobile UI and UX BehavioursBy the end of this unit, learners will be able to identify mobile-specific UI behaviors, including platform-driven rendering, list-view transformations, orientation changes, theming constraints, and other design considerations unique to mobile environments.Cert-Critical
Unit 4Differentiate Between Genero Mobile Application ArchitecturesBy the end of this unit, learners will be able to distinguish between standalone, partially connected, client-server, and development-mode applications, and identify appropriate use cases for each architecture within a mobile solution.Cert-Critical
Unit 5Describe How Genero Mobile Manages Local Data Using SQLiteBy the end of this unit, learners will be able to explain how SQLite is used as the local database, describe how it is created or packaged within the application, and the constraints and implications of using SQLite on mobile devices.Supporting
Unit 7Identify and Configure Essential Environment VariablesBy the end of this unit, learners will be able to identify key environment variables required for running Genero Mobile applications and demonstrate how to define and manage these variables within Genero Studio for correct runtime behaviour.Cert-Critical
Unit 8Explain Mobile Security ConsiderationsBy the end of this unit, learners will be able to explain how Android's sandbox model affects data storage, identify the risks of storing files outside the sandbox, and describe how to apply secure storage practices within mobile applications.Supporting
Unit 11Apply Concepts by Building a Simple Genero Mobile ApplicationBy the end of this unit, learners will be able to design, build, and test a basic Genero Mobile application that incorporates UI screens, SQLite storage, at least one device feature, and a simple interaction with a simulated or real backend service.Practical
Unit 12Genero Web AppsExplain the constraints of Genero Mobile applications. Highlight that the main things achievable in mobile can also be done in Genero Web Apps. Understand when to recommend web over mobile for a given use case.Supporting
